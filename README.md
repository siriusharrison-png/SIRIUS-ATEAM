# AI Team

转转的 AI 团队展示页面。

## 访问地址

https://siriusharrison-png.github.io/ai-team/

## 功能

- 像素风格的 Agent 形象展示
- Tag 切换查看各成员详情
- WORKFLOW 按钮查看协作流程图
- LOG 按钮查看各 Agent 更新日志

## 团队成员

| Agent | 角色 | 状态 |
|-------|------|------|
| 小秘书 | 最终汇报人 | 仅本地 |
| 设计师 | 设计系统维护 | 开源 |
| Figma 设计员工 | Figma 设计操作 | 仅本地 |
| 知识管理 | 学习知识管理 | 开源 |
| 测试QA | 设计还原度检查 | 开源 |
| 摄影师 | Unsplash 数据追踪 | 仅本地 |

## 更新日志

在 `index.html` 顶部的 `agentData` 对象中维护各 Agent 的状态和日志。
